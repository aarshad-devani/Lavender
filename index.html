What is Lavender?

Lavender is a Discord bot made by Discord lovers for Discord lovers. It includes everything you'd need from moderation tools to fun little gadgets that'll pass the time.

The Best Features

Previously Fast Servers. Coming Soon: Even Faster Services

We started as a self-host and realised that sucked. So, we tried Heroku and it was pretty nice. Now that sucks. So, coming in the future: self-hosting at a faster degree.

Quality Commands for Quality Servers

We know you don't want useful features inside you server. So, we made sure to focus on super-upper-duper-smuper fancy-smancy features over games and fun stuff. And, if you don't like the few fun things we give you: you can always disable it on the upcoming dashboard.

Open-Source Security

We hate closed-source things: it hides things we may not want to see like trackers or viruses. Lavender is completely open-source, minus the API keys, to allow everyone to enjoy the code and help it improve. So, no need to fret about your data being mined from us: cause' we can't if we tried.
